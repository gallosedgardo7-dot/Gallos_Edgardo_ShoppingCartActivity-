## Part 2 Project Description

This project is an **enhanced console-based Shopping Cart System** developed in C#. It simulates a more complete store experience where users can browse products, search and filter items by category, and fully manage their cart before checkout.

The system includes advanced features such as updating item quantities, removing items, and clearing the cart. During checkout, it validates user payment, applies discounts when applicable, and generates a detailed receipt containing the receipt number, date and time, purchased items, totals, and change.

Additionally, the program tracks completed transactions through an order history system and displays low stock alerts after checkout to help monitor inventory levels. Overall, the project demonstrates the use of arrays, methods, validation techniques, and basic object-oriented programming in building a functional shopping system.
